# React File Upload

### Technologies

The following technologies were used for this project:

* Node
* Express
* Express-Fileupload
* React
* Axios
* Concurrently

### Instructions

Download or clone this repo. From inside the project directory, open Terminal and run:
```
npm install
```

Next, change directories to the "client" directory and run:
```
npm install
```
